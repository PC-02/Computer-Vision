# Basic Motion Detection & Facial Detection

# Requirements
```
pip install -r req.txt
//Or 
pip3 install -r req.txt
```

# How to Run 

For Motion Detection:
```
python3 detect.py

OR

python3 detect2.py
```

For Facial Recognition:
```
python3 face_detect.py haarcascade_frontalface_default.xml
```
